### Project Overview
---
A comprehensive SaaS platform that allows users to generate, manage, and track professional invoices. It features a unique **AI-driven generation** capability where users can create detailed invoices using natural language prompts.

### Core Features to Highlight
---
- **AI-Powered Generation**: Integrated **Google Gemini AI** to transform natural language prompts into structured invoice data automatically.

- **Complete Invoice Lifecycle**: Features for creating, previewing, editing, printing, and saving invoices as **PDFs**.

- **Dashboard & Analytics**: A data-driven dashboard tracking total revenue, unpaid amounts, and invoice status distribution.

- **User Authentication & Security**: Implemented secure login/signup and session management using Clerk, including Google OAuth integration.

- **Business Profile Management**: Personalized business settings including branding, tax percentages, digital stamps, and signatures that auto-fill into new invoices.

- **Dynamic Data Management**: Full CRUD operations with filtering by invoice number, client name, and status (paid/unpaid).

### Technical Stack 
---
- **Frontend**: React.js, Vite, Tailwind CSS (for responsive UI/UX).

- **Backend**: Node.js, Express.js.

- **Database**: MongoDB (using Mongoose for data modeling).

- **AI Integration**: Google Gemini AI API.

- **Authentication**: Clerk (Multi-factor and Social Auth).
